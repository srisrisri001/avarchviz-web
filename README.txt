AV Archviz Website

How to launch:
1. Open index.html directly in your browser.
2. Or double-click launch-website.bat on Windows.

Main files:
- index.html: Website structure and page content.
- styles.css: Design, colors, layout, and mobile view.
- script.js: Navigation, project pages, gallery, 360 rotate, heart like count, and mobile swipe.
- assets/: Project images.
- icons/: Website icon files.

How to add your own images:
1. Copy your images into the assets folder.
2. Open index.html and replace image paths like assets/forest-villa.png.
3. Open script.js and replace project image paths inside the projects array.

Recommended image formats:
- .jpg, .png, .webp

Recommended image names:
- forest-elevation.jpg
- forest-bedroom.jpg
- azure-kitchen.webp
- terra-pool.png
